# NUCLEO-H563ZI Starter Code

This project is a *copy* of the RT-XHAL-STM32H563ZI-NUCLEO144 demo by Giovanni
Di Sirio, to simply see how to compile and upload programs onto the
NUCLEO-H563ZI. The self-test functionality has been stripped in favour of
just implementing a blink program.

The `Makefile` has been stripped of most ChibiStudio-generated elements and
re-written to be manually compiled using the `make` command.
